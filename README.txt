INTRO
Show me the weather is an app that shows you the current weather where you are.
No frills, no fuss... just the weather.

To use, make sure that you have GPS enabled and/or network services enabled.




BUILDING
This project was built using IntelliJ Idea with Gradle. Import the top level weather folder into Eclipse. It is encouraged that the libs be downloaded via gradle prior to building.
Or, use IntelliJ and choose "File > Import Project".



EXTERNAL LIBRARIES
This app could not have been developed without the following libs:

*WeatherLib
*Markushi's CircleButton
*Floating Action Button
*Materialish Progress
